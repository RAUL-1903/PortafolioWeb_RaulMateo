Conclusión

Durante el primer consolidado de la asignatura de desarrollo de aplicaciones web, se a logrado realizar avances significativos, en términos de HTML y CSS, aprendimos a crear páginas web con un diseño responsivo, lo cual implica que se adaptan de manera óptima a diferentes dispositivos y tamaños de pantalla, además, adquirimos conocimientos en JavaScript, lo que nos ha permitido animar objetos y elementos en páginas web, brindándoles interactividad y dinamismo, también exploramos el uso de Canvas, lo cual nos proporciono la capacidad de dibujar objetos y figuras, por último, pero no menos importante, incursionamos en el mundo de Node.js, aprendiendo a desarrollar aplicaciones web que se ejecutan en el lado del servidor. En general, los resultados obtenidos en este primer consolidado fueron satisfactorios y es suficiente motivación para continuar con el aprendizaje, con dedicación y esfuerzo.

Mi enfoque constante en la metacognición me permitió adaptarme a los cambios y mejorar mi capacidad para adquirir y aplicar nuevos conocimientos.

Mi enfoque constante en la metacognición me permitió adaptarme a los cambios y mejorar mi capacidad para adquirir y aplicar nuevos conocimientos.

Semana 03

¿Qué aprendí?
Durante esta semana, aprendí sobre JavaScript, eventos, manipulación del DOM y validación de formularios, adquirí conocimientos sobre el lenguaje de programación JavaScript y su papel en la creación de interactividad y dinamismo en las páginas web, también exploré cómo utilizar eventos para responder a las acciones del usuario, manipular el DOM para cambiar contenido y estilos, y validar formularios para asegurar datos correctos.


¿Cómo aprendí?

Para adquirir estos conocimientos, seguí un enfoque práctico y de autodidacta, realicé ejercicios y prácticas para implementar los conceptos aprendidos, además, consulté recursos en línea, como documentación y tutoriales, para ampliar mi comprensión y resolver dudas, aproveché la disponibilidad de recursos digitales y la comunidad en línea para acceder a información actualizada y compartir conocimientos con otros desarrolladores, a través de este proceso de aprendizaje activo y continuo, logré mejorar mis habilidades en el desarrollo web con JavaScript.

Semana 04

¿Qué aprendí?
A través de mi proceso de aprendizaje, adquirí conocimientos sobre la historia y características de JavaScript, me familiaricé con su origen en los años 90 y su propósito de brindar interactividad a las páginas web, también comprendí cómo se incluye en el código HTML mediante etiquetas de script, así como los comentarios en JavaScript y PHP, además, adquirí conocimiento sobre la declaración de variables y los diferentes tipos de datos.


¿Cómo aprendí?

Para adquirir estos conocimientos, realicé una investigación exhaustiva sobre la historia y características de JavaScript, consulté diversas fuentes, como documentación oficial, tutoriales en línea y libros especializados, además, participé en cursos en línea y realicé ejercicios prácticos para aplicar y reforzar lo aprendido, a medida que avanzaba en mi estudio, también fui capaz de reflexionar sobre los conceptos y relacionarlos con mis conocimientos previos sobre programación, lo que me permitió tener una comprensión más sólida de JavaScript.

Semana 05

¿Qué aprendí?
Aprendí cómo utilizar el elemento canvas de HTML5 junto con JavaScript para dibujar gráficos y crear animaciones en una página web, aprendí a dibujar formas básicas como líneas, rectángulos y círculos utilizando los métodos proporcionados por el elemento canvas, también aprendí a crear animaciones utilizando el método setInterval() para actualizar el dibujo en intervalos regulares, además, aprendí sobre el contexto de dibujo, las coordenadas en el canvas y cómo utilizar transformaciones para aplicar rotaciones, escalas y traslaciones a los dibujos.


¿Cómo aprendí?

Aprendí estos temas mediante la investigación y el estudio de recursos en línea, como documentación oficial, tutoriales y ejemplos prácticos, también utilicé ejercicios y proyectos prácticos para aplicar lo aprendido y fortalecer mi comprensión de los conceptos, exploré diferentes ejemplos de código y realicé experimentos propios para entender cómo funcionaba el elemento canvas y cómo utilizarlo para dibujar gráficos y animaciones, además, reflexioné sobre mis conocimientos previos sobre HTML y JavaScript para establecer conexiones y ampliar mi comprensión de estas tecnologías.

Semana 06

¿Qué aprendí?
Aprendí a crear componentes en React, que son unidades independientes y reutilizables de la interfaz de usuario que encapsulan la lógica y la presentación de una parte de la aplicación, también aprendí a utilizar el estado en React para representar la información relevante de un componente en un momento dado, por otro lado conocí la importancia de las herramientas y bibliotecas disponibles en el para facilitar el desarrollo y el mantenimiento de aplicaciones, conocí la unidireccionalidad en React, donde los datos fluyen en una sola dirección lo que facilita la comprensión y el mantenimiento.


¿Cómo aprendí?

Aprendí sobre el enfoque declarativo y la unidireccionalidad en React a través de la lectura de documentación oficial, así como de tutoriales y ejemplos prácticos. Exploré ejemplos de código que demostraban cómo React simplifica la creación de interfaces de usuario al permitirnos pensar en términos de componentes y estados. También realicé ejercicios prácticos donde pude experimentar con la creación de componentes declarativos en React y observar cómo los cambios en el estado afectaban automáticamente la representación de la interfaz de usuario.

Semana 09

¿Qué aprendí?
Comprendí cómo funciona el desarrollo backend a nivel empresarial. Aprendí a configurar servidores web usando Apache Tomcat y a estructurar aplicaciones robustas mediante Jakarta EE. Entendí la importancia de los Servlets para procesar peticiones HTTP y cómo se gestiona el ciclo de vida de una aplicación web desde el servidor.


¿Cómo aprendí?

Mi aprendizaje se centró en la configuración y el despliegue. Practiqué creando proyectos dinámicos en Java, generando archivos .war y desplegándolos manualmente en el servidor. Solucionar los errores de conexión y puertos fue clave para entender la infraestructura que sostiene a las aplicaciones web.

Semana 10 y 11

¿Qué aprendí?
Di un salto cualitativo hacia la construcción de APIs modernas. Aprendí a utilizar el framework Spring Boot para acelerar el desarrollo, implementando una arquitectura en capas que separa la lógica de negocio de los datos.También dominé la conexión a bases de datos MySQL y la importancia de validar cada "endpoint" usando herramientas profesionales.


¿Cómo aprendí?

Adopté el rol de un desarrollador backend real. Construí una API completa desde cero y utilicé Postman intensivamente para actuar como "cliente", enviando datos de prueba y verificando que el servidor respondiera correctamente (códigos 200, 404, 500), lo que me dio una visión clara del flujo de la información.

Semana 12

¿Qué aprendí?
Descubrí la agilidad del desarrollo web moderno con PHP y Laravel. Aprendí a utilizar su potente ORM (Eloquent) para manejar bases de datos sin escribir SQL puro y comprendí el patrón Modelo-Vista-Controlador (MVC). Me sorprendió lo rápido que se puede implementar un sistema de autenticación y rutas comparado con otros lenguajes.


¿Cómo aprendí?

A través de la práctica directa construyendo un sistema CRUD completo. La instalación de herramientas como Composer y XAMPP fue el primer paso, seguido de la creación de migraciones de base de datos. Comparar este flujo de trabajo con el de Java me ayudó a entender las ventajas de cada tecnología según el proyecto.

Semana 13

¿Qué aprendí?
Inicié mi camino en Python Web aprendiendo la filosofía de los "microframeworks" con Flask. Aprendí que a veces menos es más: Flask me permitió levantar un servidor funcional con muy pocas líneas de código, dándome control total sobre cómo se procesan las rutas y cómo se renderizan las plantillas HTML con Jinja2.


¿Cómo aprendí?

Me enfoqué en "aprender haciendo" prototipos rápidos. En lugar de configuraciones complejas, escribí el código de enrutamiento manualmente para entender la lógica base del backend. También practiqué la creación de entornos virtuales en Python, algo fundamental para mantener el orden en las dependencias del proyecto.

Semana 14

¿Qué aprendí?
Esta fue la semana más innovadora. Aprendí a integrar Inteligencia Artificial generativa dentro de una aplicación web tradicional. Utilicé Django para la estructura robusta y logré conectar modelos de lenguaje (LLM) locales mediante Ollama, creando un sistema capaz de entender y responder preguntas en lenguaje natural.


¿Cómo aprendí?

Fue un desafío de integración técnica. Tuve que investigar cómo comunicar el backend de Python con la API de Ollama. Realicé múltiples pruebas enviando "prompts" desde mi código y ajustando la respuesta de la IA para que fuera útil en la web, logrando fusionar la programación clásica con las nuevas tecnologías de IA
