Duodécima semana

PHP y Laravel

¿Qué temas aprendió?

Descubrí la agilidad del desarrollo web moderno con PHP y Laravel. Aprendí a utilizar su potente ORM (Eloquent) para manejar bases de datos sin escribir SQL puro y comprendí el patrón Modelo-Vista-Controlador (MVC). Me sorprendió lo rápido que se puede implementar un sistema de autenticación y rutas comparado con otros lenguajes.

Definiciones

Laravel

Framework de PHP con una sintaxis elegante y expresiva, diseñado para facilitar tareas comunes de desarrollo web.

Eloquent ORM

Mapeador objeto-relacional de Laravel que permite interactuar con la base de datos usando objetos PHP en lugar de SQL.

MVC

Modelo Vista Controlador. Patrón de arquitectura que separa los datos, la interfaz de usuario y la lógica de control.

Composer

Herramienta para gestión de dependencias en PHP, esencial para instalar Laravel y sus librerías.

Procedimientos

  1. Instalación de entorno: Configuración de herramientas base como Composer y XAMPP.
  2. Creación de migraciones: Definición de la estructura de la base de datos mediante código PHP.
  3. Desarrollo de CRUD: Implementación completa de Crear, Leer, Actualizar y Eliminar datos.
  4. Comparativa técnica: Análisis del flujo de trabajo en PHP vs Java.

Resultados

  1. Desarrollo de un sistema funcional con autenticación y rutas en tiempo récord.
  2. Gestión eficiente de bases de datos utilizando Eloquent ORM.
  3. Comprensión clara de las ventajas de agilidad que ofrece PHP/Laravel para ciertos proyectos.
  4. Implementación correcta del patrón MVC en una aplicación web real.
