Decimotercera semana

Python y Flask

¿Qué temas aprendió?

Inicié mi camino en Python Web aprendiendo la filosofía de los "microframeworks" con Flask. Aprendí que a veces menos es más: Flask me permitió levantar un servidor funcional con muy pocas líneas de código, dándome control total sobre cómo se procesan las rutas y cómo se renderizan las plantillas HTML con Jinja2.

Definiciones

Flask

Microframework web escrito en Python. Es ligero y no requiere herramientas o librerías particulares.

Entorno Virtual

Herramienta para crear entornos de Python aislados, evitando conflictos entre dependencias de proyectos.

Jinja2

Motor de plantillas para Python que permite intercalar lógica de programación dentro de archivos HTML.

Enrutamiento

Mecanismo para mapear URLs específicas a funciones de código que manejan esas solicitudes.

Procedimientos

  1. Aprender haciendo: Enfoque en prototipado rápido y codificación manual.
  2. Gestión de entornos: Creación y activación de entornos virtuales en Python.
  3. Enrutamiento manual: Escritura explícita de rutas para entender la lógica base del backend.
  4. Renderizado: Integración de plantillas HTML dinámicas usando Jinja2.

Resultados

  1. Capacidad para levantar un servidor funcional con código minimalista.
  2. Comprensión profunda de cómo se procesan las rutas web sin "magia" automática.
  3. Entendimiento de la importancia de mantener las dependencias del proyecto ordenadas y aisladas.
  4. Creación exitosa de prototipos web rápidos y ligeros.
