Decimocuarta semana

Web Inteligente e IA

Desarrollada del [DIA] al [DIA]

¿Qué temas aprendió?

Esta fue la semana más innovadora. Aprendí a integrar Inteligencia Artificial generativa dentro de una aplicación web tradicional. Utilicé Django para la estructura robusta y logré conectar modelos de lenguaje (LLM) locales mediante Ollama, creando un sistema capaz de entender y responder preguntas en lenguaje natural.

Definiciones

Django

Framework web de alto nivel en Python que fomenta el desarrollo rápido y un diseño limpio y pragmático.

Ollama

Herramienta que permite ejecutar modelos de lenguaje grandes (LLMs) localmente en tu propia máquina.

LLM

Modelo de Lenguaje Grande. IA entrenada con enormes cantidades de texto para entender y generar lenguaje humano.

Prompt Engineering

Técnica de diseñar inputs (instrucciones) para guiar a la IA y obtener la respuesta deseada.

Procedimientos

  1. Integración técnica: Investigación y comunicación entre el backend de Python y la API de Ollama.
  2. Pruebas de Prompting: Envío de múltiples "prompts" desde el código para calibrar la IA.
  3. Ajuste de respuestas: Refinamiento de la salida de la IA para que sea útil en un contexto web.
  4. Fusión tecnológica: Combinación de programación web clásica con nuevas tecnologías generativas.

Resultados

  1. Creación de un sistema web capaz de responder preguntas en lenguaje natural.
  2. Éxito en la conexión de modelos LLM locales con una aplicación Django.
  3. Comprensión práctica de cómo integrar IA en flujos de trabajo de desarrollo tradicionales.
  4. Desarrollo de habilidades en la ingeniería de prompts aplicada al backend.
